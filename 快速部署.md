# 🚀 快速部署到公网

## 最简单的方法：Vercel（推荐）

### 1. 推送到 GitHub

```bash
# 初始化 git（如果还没有）
git init

# 添加所有文件
git add .

# 提交
git commit -m "AI News Timeline"

# 在 GitHub 创建新仓库，然后推送
git remote add origin https://github.com/你的用户名/仓库名.git
git branch -M main
git push -u origin main
```

### 2. Vercel 部署（2分钟搞定）

1. 访问 [vercel.com](https://vercel.com) 并注册/登录
2. 点击 "Add New Project"
3. 选择刚才的 GitHub 仓库
4. 点击 "Import"
5. **直接点击 "Deploy"**（默认配置即可）
6. 等待 1-2 分钟

### 3. 完成！🎉

Vercel 会给你一个地址，类似：

`https://ai-news-timeline-xxxx.vercel.app`

把这个链接发给朋友，任何人都可以访问了！

## 遇到问题？

如果 Vercel 无法自动识别配置，可以：

1. 在 Vercel 项目设置中：
   - **Framework Preset**: Vite
   - **Root Directory**: ./
   - **Build Command**: npm run build
   - **Output Directory**: dist

2. 然后重新部署
